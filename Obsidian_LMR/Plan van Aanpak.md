**Doorlooptijd**: ~20 dagen (4 weken) (nog wat unknowns die doorlooptijd kunnen beïnvloeden, BE is nog niet in rekening gebracht)
**Team**: 2 Frontend + 1 Backend + 1 FA + 1 PM

## Team Rollen

- **Frontend Dev 1**: components (Level/Job Selection, Quiz, UI, animaties, contexts)
- **Frontend Dev 2**: components (Level/Job Selection, Quiz, UI, animaties, contexts)
- **Backend Dev**: APIs, database, CMS integratie
- **FA**: Requirements, testing, kwaliteitscontrole
- **PM**: Planning, communicatie, risicobeheer

## Planning (4 weken)

| Week  | Focus    | Frontend                    | Backend              | Resultaat            |
| ----- | -------- | --------------------------- | -------------------- | -------------------- |
| **1** | Opzetten | Basis project + componenten | API + database       | Werkende basis       |
| **2** | Selectie | Level keuze + kaart         | Content API's        | Selectie schermen    |
| **3** | Quiz     | Vragen + timer              | Validatie + tracking | Quiz functionaliteit |
| **4** | Afronden | Resultaten + testen         | Security + productie | Klaar product        |

## Workflow & Risico's

**Daily Standups**: 15min - voortgang, planning, blockers
**Code Review**: PR's binnen 4u, pair programming, API contract reviews

**Risico's**: Animatie complexiteit, performance, browser support, API performance, data consistency, security
**Mitigatie**: Early prototyping, performance monitoring, fallbacks, caching, indexing, security audits

## Success Criteria

✅ **Functioneel**: End-to-end quiz flows, smooth animaties, accurate results, cross-browser support
✅ **Technisch**: 80% code coverage, Lighthouse >90, accessibility, mobile responsive, API <200ms, security
✅ **Business**: Bug-free user journey, acceptable performance, scalable architecture
